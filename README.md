# SEXY TIPS — сайт бренда

Нишевый сайт для бренда **SEXY TIPS** (субкультура секси типов) — гранж / трэш /
техно-андеграунд мерч из Москвы. Весь контент (фото, тексты, цены, ивент, дресс-код)
взят с телеграм-канала [@ssexytips](https://t.me/ssexytips).

## Страницы
- **index.html** — главная: intro-гейт → герой → манифест → дроп «Бэнгер» →
  лукбук → CTA → футер.
- **catalog.html** — каталог: фильтры по категориям + 7 товаров + быстрый просмотр
  (модалка с размерами, цветами, спекой и кнопкой заказа).

## Как открыть
Это статический сайт без сборки. Открыть можно двумя способами:
1. Сервер с приёмом заказов (рекомендуется): `python3 server.py` → http://localhost:8000.
   Оформленные на сайте заказы (номер, контакт, товары) дописываются в `orders.json`.
2. Просто открыть `index.html` в браузере (заказы тогда остаются только в localStorage).

## Структура
```
index.html, catalog.html
assets/
  css/style.css      — вся дизайн-система и компоненты
  js/site.js         — общие эффекты (курсор, грейн, гейт, reveal, маркизы, прогресс)
  js/catalog.js      — данные товаров + рендер каталога + модалка
  img/               — фото с канала (tg_*) + обработанный герой (hero_blue.jpg)
  tex/               — текстуры: зерно (noise.png), сетка (mesh.png)
```

## Визуальные решения
Хром-типографика, глитч, плёночное зерно, скан-линии, кастомный курсор,
бегущие строки (техно-тикеры), магнитные кнопки, scroll-reveal, дуотон-обработка фото,
intro-гейт «войти в тусу», прогресс-скролл и вертикальные сайд-рейлы.
Палитра: **серебро / хром / блеск / сетка / голубой / чёрный** + розовый мерча.
